# 回归检查清单

用于发版或改动编辑器核心流程前后的快速自检。当前没有自动化测试，至少执行 `npm run build`，涉及 UI 时再打开 `http://127.0.0.1:5173/` 手动验证。

## 基础验证

- 运行 `npm run build` 成功。
- 打开首页和编辑器，没有控制台报错或明显布局错位。
- 新建默认印章后，默认主色为 `#FF0015`。

## 画布与元素

- 切换常用模板后，画布、标尺、网格正常显示。
- 拖拽、缩放、选中元素时，蓝色选框位置与实际元素一致。
- 左侧元素列表只显示已有元素；点击元素后，右侧属性面板能同步到对应设置。
- 删除当前选中元素后，选中态能移动到相邻元素或正确清空。

## 属性面板

- 基础设置和高级设置切换正常。
- 公司文字、印章类型、编码、中间文字支持编辑，多行内容能正常绘制。
- 字体选择保持可用，尤其是华文隶书和本地小标宋相关选项。

## 图片提取

- 上传或拖拽非敏感样例图片后，能自动生成透明 PNG 结果。
- 透明背景、边缘增强、保留深浅、输出颜色调整都能即时刷新。
- 重新上传、清除结果、替换画布流程正常。
- 默认输出颜色为 `#FF0015`。

## 导出预览

- 打开下载弹窗时，导出预览与画布的标尺、网格状态保持一致。
- 关闭下载弹窗后，画布标尺和网格状态不丢失。
- PNG 透明背景、白底 PNG、JPEG 等常用导出路径可生成文件。

## 发布前

- `git status --short` 中只 stage 本次相关文件。
- 不提交真实印章、合同、发票、身份资料或敏感样例图片。
- 不误删或误提交无关的 `scripts/` 目录。
