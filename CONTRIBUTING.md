# Contributing to DrawStamp Studio

感谢你愿意参与 DrawStamp Studio 的改进。

这个项目的目标是做一个轻量、实用、可信赖的在线电子印章编辑器。提交功能或问题时，请尽量围绕真实使用场景：生成印章、提取印章、编辑排版、模板复用、导出图片和本地隐私处理。

## 开发环境

推荐使用 Node.js 20 或更高版本。

```bash
npm install
npm run dev
```

本地开发地址通常是：

```text
http://127.0.0.1:5173/
```

提交前请至少运行一次：

```bash
npm run build
```

## 贡献方式

- 提交 bug 时，请说明浏览器、系统、复现步骤、期望效果和实际效果。
- 提交 UI 优化时，请尽量附截图或录屏，说明调整前后的差异。
- 提交新模板时，请说明模板适用场景，例如合同、财务、发票、收讫、审核等。
- 提交图片提取相关问题时，请尽量使用不含敏感信息的测试图片。
- 不要提交真实公司印章、合同、票据、身份证明等敏感材料。

## 代码风格

- 尽量沿用现有 Vue 组件和工具函数结构。
- UI 保持克制、清晰、实用，不做过度装饰。
- 和印章绘制相关的逻辑优先放在 `src/utils/Draw*.ts` 或现有绘制模块中。
- 和编辑器界面相关的逻辑优先放在 `src/components/editor/` 中。
- 保持处理流程本地优先，不引入不必要的后端依赖。

## Pull Request 建议

提交 PR 时，建议包含：

- 本次修改解决的问题。
- 主要修改点。
- 已验证的命令或浏览器行为。
- 如涉及界面变化，请附截图。

PR 标题可以使用：

```text
fix: 修复某个编辑器问题
feat: 增加某个印章模板
docs: 更新项目文档
style: 优化界面样式
```

## 安全与合规

DrawStamp Studio 仅用于学习、测试和合规场景下的电子印章图片制作。请勿提交或推广任何用于违法、欺诈、伪造公文、伪造合同、伪造票据的功能或示例。
