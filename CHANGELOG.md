# Changelog

本项目的版本记录遵循简洁的人工维护格式。

## 0.2.5 - 2026-06-12

### Added

- 增加 DrawStamp Studio 项目定位、README 截图、SEO 信息和社交预览图。
- 增加图片提取印章能力，支持拖拽上传、本地处理、透明 PNG 输出。
- 增加专业画布工具栏：缩放、适配窗口、重置视图、背景切换和导出区域。
- 增加常用模板入口和模板导入导出能力。
- 增加基础设置与高级设置分区，让属性面板更清晰。
- 增加内置华文隶书字体资源。

### Changed

- 优化编辑器三栏布局，弱化重复入口，左侧元素列表更专注于选择对象。
- 优化画布选中框、拖拽移动、中心辅助线和边界提示。
- 默认印章颜色调整为更稳定的红色。
- 导出体验调整为更接近专业编辑器的底部导出区域。

### Fixed

- 修复适配窗口后印章和选择框偏移的问题。
- 修复提取印章后默认印章残留、位置调整异常等问题。
- 修复圆形章编码显示异常的问题。
- 修复字体选择中部分中文字体不可用的问题。

## 0.2.0 - 2026-06-11

### Added

- 完成 Cloudflare Pages 静态部署配置。
- 支持在线访问和 GitHub 仓库发布。
- 增加项目公开展示所需的 README、截图和部署说明。

## 0.1.0

### Added

- 基础电子印章绘制能力。
- 支持 Canvas 绘制、文字元素、图形元素和 PNG 导出。
