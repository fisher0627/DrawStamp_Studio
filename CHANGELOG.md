# Changelog

本项目的版本记录遵循简洁的人工维护格式。

## Unreleased

## 0.6.1 - 2026-07-20

### Added

- 新增中英文界面切换，根据浏览器语言自动选择，并在本地记住用户选择。
- 新增完整英文 README，GitHub 仓库首页提供中英文文档入口。

### Changed

- 优化中英文 SEO 标题、页面描述、关键词、Open Graph、Twitter Card、HTML 语言声明和 JSON-LD 结构化数据。
- 完善导出、模板、属性、图片提取、关于、隐私、条款和联系页的英文文案。

## 0.6.0 - 2026-07-09

### Added

- 新增 DrawStamp Studio 印石风格品牌 logo，并同步提供站点 logo、favicon、PWA 图标、Apple Touch Icon 和社交预览图。
- 在首页 SEO 内容区、编辑器顶部和说明页侧栏增加克制的品牌露出，让 GitHub、搜索结果和网站界面保持一致。

### Changed

- 更新 SoftwareApplication 结构化数据、manifest、Open Graph 图片和 GitHub 展示素材中的品牌图像。
- 将项目版本更新为 `0.6.0`，作为 logo 与品牌系统升级版本。

## 0.5.0 - 2026-07-09

### Changed

- 升级 DrawStamp Studio 设计系统，进一步统一编辑器的纸感工作台、工具分组、状态区域和响应式细节。
- 同步 GitHub 仓库展示信息，将 README、package 元数据、结构化数据和社交预览源文件统一到 `DrawStamp_Studio` 新仓库地址。
- 更新 SEO 元数据与 sitemap 日期，保持公开站点、GitHub 仓库首页和搜索引擎入口一致。
- 补充 README 英文项目摘要，提升 GitHub 英文搜索和外部引用识别度。

### Fixed

- 修复元素列表删除或撤销后，画布绘制工具内部配置可能覆盖最新 store 配置的问题。
- 修复 GitHub 旧仓库地址在结构化数据、反馈链接和社交预览源文件中残留的问题。

## 0.4.0 - 2026-06-19

### Added

- 增加更完整的 GitHub 展示素材，更新 README 截图与社交预览图。
- 补齐多语言界面文案，避免字体样式和加载状态出现缺失 key。
- 增加首页 SEO 内容区、FAQ 结构化数据、基础说明路由和 Cloudflare Pages SPA 回退配置。
- 增加 Telegram 联系二维码，并将说明页联系方式统一为 Telegram 与 `fisherztz@gmail.com`。
- 增加 `InfoPageShell` 项目档案页框架，统一关于项目、隐私政策、服务条款和联系反馈页面。

### Changed

- 重塑编辑器视觉系统，统一为更克制、专业的纸感工作台风格。
- 优化顶部工具栏、模板库、画布工具分组、导出弹窗、图片提取弹窗和属性面板质感。
- 统一默认印章红色为 `#FF0015`，并收敛按钮、选中态、导出预览等颜色层级。
- 图片提取入口文案调整为“提取印章”，让功能意图更直接。
- 更新 sitemap、结构化数据、README 部署说明和基础安全响应头，保持 GitHub 展示与线上 SEO 信息一致。
- 清理关于、隐私、条款、联系页面中的旧项目名称、旧联系方式和旧邮箱信息。
- 统一“关于项目 / 联系反馈”入口命名，并将联系页反馈按钮改为“打开邮件草稿”。
- 优化说明页移动端导航和首页 SEO 内容区，让公开页面与编辑器质感更一致。

### Fixed

- 修正导出弹窗关闭后标尺、网格等画布状态容易被视觉流程干扰的问题。
- 清理旧版全局按钮样式残留，避免新旧界面风格混用。
- 修复 Vue I18n 将 Telegram 账号和邮箱中的 `@` 解析为 linked message 导致控制台报错的问题。

## 0.3.0 - 2026-06-18

### Added

- 增加 DrawStamp Studio 项目定位、README 截图、SEO 信息和社交预览图。
- 增加图片提取印章能力，支持拖拽上传、本地处理、透明 PNG 输出。
- 增加专业画布工具栏：缩放、适配窗口、重置视图、背景切换和导出区域。
- 增加常用模板入口和模板导入导出能力。
- 扩展模板库，新增公司公章、收讫专用章、业务专用章、报价专用章，并支持模板分类筛选。
- 增加基础设置与高级设置分区，让属性面板更清晰。
- 增加内置华文隶书字体资源。
- 增加本地自动草稿，保留最近 5 个版本，刷新页面后可继续当前编辑状态。

### Changed

- 优化编辑器三栏布局，弱化重复入口，左侧元素列表更专注于选择对象。
- 优化画布选中框、拖拽移动、中心辅助线和边界提示。
- 默认印章颜色调整为更稳定的红色。
- 导出体验调整为更接近专业编辑器的底部导出区域和下载弹窗，支持格式卡片、实时预览、倍数选择、白底 PNG 与高级尺寸设置。

### Fixed

- 修复适配窗口后印章和选择框偏移的问题。
- 修复提取印章后默认印章残留、位置调整异常等问题。
- 修复圆形章编码显示异常的问题。
- 修复字体选择中部分中文字体不可用的问题。
